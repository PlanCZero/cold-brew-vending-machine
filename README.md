# Getting Started
```sh
git clone https://github.com/PlanCZero/cold-brew-vending-machine.git
cd vending-machine
npm i
npm start
```

# Description
Cold Brew vending machine is a proof-of-concept project that shows how React connects with Redux to manage the state of a component.
